## Issue 3: LiveStore schema — Projects table + events + materializer — COMPLETED

Date: 2026-03-04

### What was done
- Defined `projects` table using `State.SQLite.table` with columns: `id` (text, primary key), `repoPath` (text), `name` (text), `rlphConfig` (text, nullable).
- Defined `projectCreated` event using `Events.synced` with schema: `{ id, repoPath, name, rlphConfig }`. Event name: `v1.ProjectCreated`.
- Defined `projectRemoved` event using `Events.synced` with schema: `{ id }`. Event name: `v1.ProjectRemoved`.
